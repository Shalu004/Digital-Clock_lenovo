# Digital Clock Using JavaScript

## Objective
Create a real-time digital clock using HTML, CSS, and JavaScript.

## Features
- Displays current hours, minutes, and seconds
- Updates automatically every second
- Responsive and clean design

## Technologies Used
- HTML
- CSS
- JavaScript

## Learning Outcomes
- Understanding JavaScript Date Object
- Using setInterval()
- DOM Manipulation
- Dynamic Web Applications
